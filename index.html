In Safari, tap the Share button (box with arrow) → "Add to Home Screen" → Add. You'll get a full-screen icon just like Duolingo!

Siri Shortcut Setup

Say "Hey Siri, Log Food"

  1. Open the Shortcuts app
  2. Tap + to create a new shortcut
  3. Add action: Dictate Text
  4. Add action: Open URLs
  5. Set URL to: YOUR_URL/?food= then tap the variable button and select the Dictate Text result
  6. Name it "Log Food"
  7. Say "Hey Siri, Log Food" — Siri records what you say and opens the app pre-filled!
